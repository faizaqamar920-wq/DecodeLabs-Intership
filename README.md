##  Project 1: Password Strength Checker (`script.py`)
#  Goal
To design and implement a verification program that evaluates a user’s password string to classify it as weak, medium, or strong based on 
safety metrics.
# ️ Implemented Requirements:
**Length Constraints:** Validates password character density thresholds.
 **Character Variety Checks:** Verifies the concurrent inclusion of numeric digits, symbols, and mixed-case letters (uppercase and lowercase) 
**Strength Diagnostics:** Processes condition checks to provide a final clear strength result.
##  Project 2: Confidentiality Logic – Caesar Cipher (`Caesar.py`)
#  Goal
To implement a operational symmetric encryption and decryption pipeline to safe-keep text data during transmission.
# ️ Implemented Requirements:
**Format Preservation:** Keeps formatting, text spaces, and non-alphabetic punctuation symbols completely intact throughout execution loops.
* **Output Stream:** Sequentially displays both the transformed encrypted text and the restored original text directly within the terminal workspace.
